# Advent of Code
This is my advent of code repository. Starting this year, 2023, I plan to do as many advent of code challenges as possibile, while learning a bunch of new things doing it. But although I got the idea from doing this in the first place while thing about advent of code. My focus is really on figuring out the things below. Any solved advent of code challenge is just an excuse to start building in the first place.

### The plan is to do at the following
* Only use my terminal for anything git. No GUIs for this one. Even this README I am currently writing in nano, so excuse any typos.
* Use [bun](bun.sh)
* Use GitHub projects for my todo list
* Create a nice design system with proper light and dark mode including a public storybook documentation based on [Vercel's Geist font](https://vercel.com/font). (Maybe trying out [Meta's StyleX](https://stylex-docusaurus.vercel.app/))
* Implement with an authentication provider
* Solve all challenges on BE and integrate via an API
* Use a feature flag service like [LaunchDarkly](https://launchdarkly.com/)
* Use GitHub Actions to measure my web-vitals and auditing the accessibility
* Deploy on vercel
* I want to document the whole process in a short blog. Maybe I keep it for myself, maybe I publish it. Who knows...

This is a lot, but I give myself a headstart. Today is November 11th. Let's see where this whole project is at Christmas.
